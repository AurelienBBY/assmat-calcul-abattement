Calcul de l’abattement fiscal – Assistante maternelle

Outil de calcul pour la déclaration des revenus avec abattement forfaitaire.

Comment fonctionne cet outil ?

Cet outil vous aide à calculer l’abattement fiscal des assistantes maternelles pour préparer votre déclaration.

Le principe est simple : vous remplissez vos jours de garde mois par mois, et l’outil calcule automatiquement l’abattement correspondant (par jour et par enfant), puis affiche un récapitulatif.

  1. Choisissez la période à compléter (année et mois).
  2. Renseignez vos jours : heures de présence, nombre d’enfants, indemnités si nécessaire.
  3. Vérifiez le récapitulatif (abattement total + montant à déclarer).
  4. Sauvegardez ou imprimez pour archiver le mois.

💡 Astuce : exportez le mois une fois terminé, puis importez-le plus tard si vous devez le modifier ou le vérifier.

Tutoriel détaillé

1) Choisir la période

  • Sélectionnez l’année et le mois.
  • Le tableau se met à jour automatiquement.

2) Remplir les jours

  • Pour chaque jour travaillé, indiquez les heures (et les enfants concernés si applicable).
  • Renseignez les indemnités / frais si ton outil les prévoit.

3) Lire le récapitulatif

  • Vérifiez le total mensuel calculé.
  • Comparez avec la fiche de paie si besoin.

4) Sauvegarde / Import

  • Exporter : enregistre un fichier JSON sur ton ordinateur.
  • Importer : recharge un fichier exporté pour reprendre un mois.

5) Impression / PDF

  • Cliquez sur Imprimer / PDF pour générer une version imprimable.
  • Dans la fenêtre d’impression : choisissez “Enregistrer en PDF”.
